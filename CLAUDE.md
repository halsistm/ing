# ∞Gallery — Infinite Exhibition

## リポジトリ
- **GitHub**: https://github.com/halsistm/ing
- push先: `git push origin main --force`（履歴が diverge している場合は `--force` 必須）

## プロジェクト概要
Three.jsを使った3Dギャラリー探索ゲーム。
プレイヤーが無限に続くギャラリー空間を歩き回り、クリーチャーを捕獲・図鑑に登録していく。

## 技術スタック
- **Three.js r128**（CDN）
- **MarchingCubes.js**（3Dオブジェクト生成）
- **GLTFLoader.js**（3Dモデル読み込み）
- 純粋なHTML/CSS/JavaScript（フレームワークなし）
- バックエンドなし、ローカル完結

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | エントリーポイント、全UI要素のHTML/CSS |
| `engine.js` | ゲームエンジン本体（レンダリング、物理など） |
| `effects.js` | ビジュアルエフェクト |
| `combat.js` | 戦闘・捕獲ロジック |
| `weapons.js` | 武器システム |
| `weapons_patches.js` | 武器のパッチ・修正 |
| `room.js` | 部屋生成・管理 |
| `room_patches.js` | 部屋のパッチ・修正 |
| `creatures.js` | クリーチャー定義・挙動 |
| `creatures3D.js` | クリーチャーの3Dレンダリング |
| `collection.js` | 図鑑（捕獲記録）システム |
| `audio.js` | 音声管理 |
| `player.js` | プレイヤー操作・状態 |
| `player_patches.js` | プレイヤーのパッチ・修正 |
| `init.js` | 初期化処理 |
| `loop.js` | ゲームループ |
| `ui.js` | UI更新・表示制御 |

## 主なUI要素
- タイトル画面（はじめから / つづきから）
- ローディング画面（進捗バー付き）
- 図鑑オーバーレイ（50体のクリーチャー）
- カード詳細モーダル
- モバイル用バーチャルスティック
- メモポップアップ
- 捕獲通知

## 対応デバイス
- PC（マウス＋キーボード）
- モバイル（タッチ操作、バーチャルスティック）

## 操作系（PC）
- クリック：ゲーム開始・照準
- TAB：カーソル切り替え
- N：次の部屋
- Z：図鑑を開く
- Shift+矢印など：移動

## 注意事項
- スクリプトのロード順が重要（index.htmlのscriptタグ順を守る）
- `creatures.js`は`room.js`の後にロードする必要あり
- セーブデータはlocalStorageを使用（つづきから機能）
- 12秒以内にロード完了しない場合はエラー表示
# 作業ルール

- 明示的に指示された箇所**のみ**変更すること
